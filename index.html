About

This device is an all-in-one electronics workbench. It uses a Raspberry Pi 4 to run all of the processes. This workbench has the ability to perform the functionalities of a Function Generator, Ohmmeter, Voltmeter, DC Reference, Frequency Measurement, and LCD Color Change.

Connecting to Power

To connect the workbench to power, using the wall adapter, plug into the USB-c plug on the Raspberry Pi. The power adaptor should also be plugged into the wall.

Controls

To navigate through the different settings, the main way to do so is by using the knob for different controls. When you rotate it counterclockwise, it goes to the left or decrements values depending on which electronic workbench device you’re using. To go right or increase values, it can be done by rotating it clockwise.

The controls also include a fast and slow increment based on the tool required. For example, when trying to change the frequency, you can spin counterclockwise quickly to increment by 1000, and slowly to increment by 100.

To make a selection you only need to click once. Finally, holding the button down for 3 seconds will return to the main menu.

Voltmeter

Overview

The Voltmeter function utilizes a Raspberry Pi to measure voltage and display the results on an LCD screen. The system supports measurements from both internal and external sources, and provides menu navigation with a rotary encoder.

How to use the Voltmeter

  1. Power on the device and ensure that the LCD and rotary encoder are properly connected.

  2. Use the rotary encoder to navigate through the main menu options. When you reach the "Voltmeter" option, press the rotary encoder to select it.

  3. You will now be presented with the "Voltmeter Source" menu. Here, you can choose between "External" and "Internal" voltage sources.

    • If you wish to measure the voltage from an external source, connect the appropriate test leads to the external input terminals and select "External."

    • If you wish to measure the voltage from an internal source, ensure that the internal voltage source is properly connected, and select "Internal."

  4. After selecting the voltage source, the Voltmeter will display the voltage measurement on the LCD screen in the format "X.XX V +/- 0.25," where "X.XX" represents the measured voltage value.

  5. If you need to switch between external and internal sources or return to the main menu, press the rotary encoder and navigate through the menu options.
