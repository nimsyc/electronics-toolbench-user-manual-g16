Voltmeter

Overview The Voltmeter function utilizes a Raspberry Pi to measure voltage and display the results on an LCD screen. The system supports measurements from both internal and external sources, and provides menu navigation with a rotary encoder. How to use the Voltmeter Power on the device and ensure that the LCD and rotary encoder are properly connected. Use the rotary encoder to navigate through the main menu options. When you reach the "Voltmeter" option, press the rotary encoder to select it. You will now be presented with the "Voltmeter Source" menu. Here, you can choose between "External" and "Internal" voltage sources. If you wish to measure the voltage from an external source, connect the appropriate test leads to the external input terminals and select "External." If you wish to measure the voltage from an internal source, ensure that the internal voltage source is properly connected, and select "Internal." After selecting the voltage source, the Voltmeter will display the voltage measurement on the LCD screen in the format "X.XX V +/- 0.25," where "X.XX" represents the measured voltage value. If you need to switch between external and internal sources or return to the main menu, press the rotary encoder and navigate through the menu options. Important Notes The Voltmeter function provides an approximate voltage measurement with a specified uncertainty of +/- 0.25 V. Before connecting or disconnecting any voltage sources, ensure that the device is powered off to prevent any potential damage to the components. Make sure to properly connect the test leads and internal voltage sources according to the manufacturer's recommendations to ensure accurate measurements. By following these instructions, users should be able to effectively use the Voltmeter function to measure voltage from both internal and external sources, and navigate through the menu options using the LCD and rotary encoder.
